# Designer 창

`Tools > NexUI > Designer`는 화면 제작의 중심 창입니다.

| 영역 | 역할 |
|---|---|
| Global Toolbar | Screen 선택, Backend 상태, Simple/Advanced 전환, Preview·Validation·Save 실행 |
| Left Sidebar | Metadata 선택, Components 추가, Layers 탐색 |
| Canvas Toolbar | 선택 도구와 Preview 표시 옵션 조정 |
| Canvas | Element 선택, 배치, 크기 조절과 박스 선택 |
| Right Inspector | Design, Prototype, Motion 속성 편집 |
| Bottom Drawer | Timeline 자리, Validation, History, Screen Graph, Preview Log |
| Command Palette | `Ctrl/Command+K` 또는 `Ctrl/Command+Shift+P`로 명령 검색 |

Simple 모드는 일반 배치에 필요한 항목을 우선 표시합니다. Advanced 모드는 Constraints, Screen Definition, Motion, Theme, Accessibility, Policy와 Capability 같은 기술 항목을 추가로 보여 줍니다.

Bottom Drawer 높이, 열린 탭, Workspace, Preview Mode와 Canvas Scroll은 EditorPrefs에 보존됩니다. Screen과 Metadata는 경로 대신 Asset GUID로 복원합니다.

> [!NOTE]
> Bottom Drawer의 Timeline은 현재 안내 영역이며 실제 Motion 편집은 `Tools > NexUI > Designer > Advanced > Motion Clip Editor`에서 수행합니다.
